=====
Using
=====

.. Define |product name| in conf.py

Include user tasks that can be completed using the product. User tasks
should match/reflect the items listed in the Features section.
Create as many tasks as necessary by using the task template.
You can organize tasks by using categories.

Tasks in the Using section are generally end-user tasks. Tasks completed
by an administrator, or maintenance tasks, go in the Administering section.

This section provides information about using the
|product name| product.
